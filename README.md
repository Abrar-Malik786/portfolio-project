# portfolio-project
## Description
This is a personal portfolio project built with **HTML** and **CSS**.  
It showcases my personal information, education, skills, hobbies, achievements, and contact details.  
Interactive hover effects are added on the header and image to make the portfolio visually appealing.
The project also includes basic **responsive design** for smaller screens.
